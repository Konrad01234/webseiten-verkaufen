# Burger Brothers — Statische Website

Drei fertige HTML-Seiten, kein Build-Schritt, kein Node.js nötig.

## So öffnest du die Website

**Ganz einfach:** Doppelklick auf `index.html` — die Seite öffnet sich in deinem Standardbrowser.

Die drei Seiten sind:
- `index.html` — Startseite mit Hero
- `menu.html` — Speisekarte mit 5 Signature-Burgern
- `kontakt.html` — Kontakt & Öffnungszeiten

## Dateien

```
burger-brothers-static/
├── index.html      ← hier doppelklicken
├── menu.html
├── kontakt.html
└── styles.css
```

## Hinweis

Die Bilder werden von Unsplash geladen, du brauchst also eine Internetverbindung beim Öffnen. Schriftarten (Bebas Neue, Inter, JetBrains Mono) kommen ebenfalls von Google Fonts.

## Download als ZIP

Auf GitHub: Branch `claude/restaurant-burger-website-JsWAo` → grüner "Code"-Button → "Download ZIP".
